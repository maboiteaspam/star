star

never forget about your preferred packages

Install

npm i @maboiteaspam/star -g

Usage

It s a command line binary to star your preferred package.

@maboiteaspam/star 1.0.0
never forget about your preferred packages

    Usage
        star [cmd] [opts]
        star add [modules...]
        star rm [modules...]
        star ll

    Options
        -v|--verbose    verbose
        -d|--dev        --save-dev
        -h|--help       show help

    Examples
        star add debug -v
        star add debug -d
        star rm debug -v
        star ll -v
        star ll -v -d
        star -v
        star -h
